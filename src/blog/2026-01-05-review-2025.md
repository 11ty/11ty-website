It was _another huge_ year for 11ty. We shipped _177 releases_ (73% more than 2024). We closed 804 issues (15% more than 2024). We reduced our [dependency count by 28% and weight by 22%](https://www.11ty.dev/blog/dependency-watch/). More folks are building with Eleventy than ever: our year-over-year npm downloads are up by 51%! **Eight years into this roller coaster of an open source project and it still feels like weâ€™re just getting started!**

## Highlights

Here is a small, curated collection of 2025 highlights:

- In April, we moved to our [own Mastodon server](https://neighborhood.11ty.dev/@11ty). Also the [amazing Eleventy LibDoc community starter project](https://eleventy-libdoc.netlify.app/) first shipped!
- In May, we shipped a sizeable [v3.1.0 release](https://github.com/11ty/eleventy/releases/tag/v3.1.0) focused on reducing dependencies and improving build performance! We also shipped an [interactive playground component to the docs](https://neighborhood.11ty.dev/@11ty/114519676689929120) (Eleventy in the browser!).
- In June, we were featured in a [video on the Veronica Explains YouTube channel](https://www.youtube.com/watch?v=4_bYUVGgQQo)!
- In July, we started [shipping 4.x alpha pre-releases](https://www.11ty.dev/docs/versions/)!
- In August, I found an [old note from my late grandmother about the origins of the Eleventy name](https://www.zachleat.com/web/history-of-eleventy-name/).
- In September, we found our [logo added to the Font Awesome v7 icon set](https://neighborhood.11ty.dev/@11ty/115226254940993358)!
- In October, we changed the docs [supporter facepile to include emeritus individual supporters](https://www.zachleat.com/web/permanent-eleventy-facepile/) and shamefully [added long-awaited copy-paste buttons](https://www.zachleat.com/web/copy-paste/).
- In November, we [rehauled how we use GitHub Issues](https://www.11ty.dev/blog/github-issues/) and made a bunch of long-overdue progress on our Issue tracker. [Estela Franco also showed us their Eleventy source code backpack](https://neighborhood.11ty.dev/@guaca@toot.cafe/115553703597032584)
- In December, we [improved our security footprint by eliminating Access Tokens from our npm package publishing process](https://www.11ty.dev/blog/npm-security/).
	- We simplified our historical docs by centralizing to [v0, v1, v2, and v3 subdomains](https://neighborhood.11ty.dev/@11ty/115663215128943589).
	- We improved the docs for [different flavors of JavaScript](https://www.11ty.dev/docs/cjs-esm/) (CommonJS, ESM, and TypeScript) to use in Eleventy (including guidance on Nodeâ€™s new type stripping features).
- With a special sustained community nod to the [Ã—17 11ty Bundle Newsletters](https://11tybundle.dev/blog/) and [Ã—8 11ty Meetups](https://11tymeetup.dev/events/) that happened in 2025!

### Major Releases

- [Eleventy Image v6.0](https://github.com/11ty/eleventy-img/releases/tag/v6.0.0) to optimize images in Eleventy projects.
- [Image Color v1.0](https://github.com/11ty/image-color/releases/tag/v1.0.0) is a new utility to fetch colors from an image. [README](https://github.com/11ty/image-color/blob/main/README.md).
- Long overdue [v1.0 of Eleventy Navigation](https://github.com/11ty/eleventy-navigation/releases/tag/v1.0.0) to create hierarchical navigation menus in Eleventy.
- Both [v6](https://github.com/11ty/eleventy-plugin-vite/releases/tag/v6.0.0) and [v7 of the Eleventy Vite Plugin](https://github.com/11ty/eleventy-plugin-vite/releases/tag/v7.0.0) continues use of Eleventy with Vite (thank you to our lovely maintainer [Kilian Finger](https://www.kilianfinger.com/) for these!)
- [Parse Date Strings v2.0](https://github.com/11ty/parse-date-strings/releases/tag/v2.0.0) is used to add RFC-9557 (Temporal API) date parsing support to Eleventyâ€™s `date` property.
- [`<is-land>` v5.0](https://github.com/11ty/is-land/releases/tag/v5.0.0) is a lazy loader for component code in any web site.
- [Eleventy Font Awesome v1.0](https://github.com/11ty/eleventy-plugin-font-awesome/releases/tag/v1.0.0) allows use of hyper-optimized tree-shaken Font Awesome SVG icons in Eleventy projects.
- Pulling some dependencies in-house for maintenance and cleanup:
	- [Recursive Copy v4.0](https://github.com/11ty/recursive-copy/releases/tag/v4.0.0) and [v5.0](https://github.com/11ty/recursive-copy/releases/tag/v5.0.0) (facilitates our passthrough copy feature)
	- [Gray Matter v2.0](https://github.com/11ty/gray-matter?tab=readme-ov-file#11tygray-matter) (and v1.0) (front matter parsing)
	- With a future nod to pre-release work done to update [Nunjucks](https://github.com/11ty/nunjucks?tab=readme-ov-file#11tynunjucks)

Most utilities and plugins listed above can be used independently of Eleventy and Eleventy projects (exceptions to the Vite and the Font Awesome plugins).
